The 10 images contain 3 different types of tumors (glioma, meningiona, pituitary, 3 images each) and 1 image with no tumor.

To download repository:
  In command line type "git clone <URL>"
  In browser click green "Code <>" button, then "Download ZIP"

Uploading files/changes:
  In browser:
    click "Add File" and then "Upload From Computer"
      If a file with the same name exists, it will be replaced with the new version
  In Command Line:
    If a file is not yet in the repository, type "git add" to add all untracked files, or "git add <filename>"
    To commit all changes to tracked files and upload newly added files, type "git commit"
      This will give a prompt to type a commit message. To include the message in the command, type "git commit -m 'message'" including the single quotes
    To add all untracked files and commit changes in a single command type "git commit -am 'message'"
